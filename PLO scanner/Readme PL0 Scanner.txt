1. change the name of the input file to the appropriate file on line 59 and 82 of PL0scanner.c
2. upload PL0scanner.c and the appropriate text file onto eustis server.
3. type "gcc PL0scanner.c" into eustis terminal.
4. type "./a.out" into eustis terminal to run the file.  Once run, it will make an "output.txt" file.
5. type "cat output.txt" to read the file.
